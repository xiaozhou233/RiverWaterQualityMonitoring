## 水质指标
pH值
| pH | 评价 |
| :---: | :---: |
| 6.5~8.5 | 优秀 |
| 6~9 | 合格 |
| <6 或 >9 | 异常 |

浊度 (Turbidity)

| 浊度 (NTU) | 评价 |
| :---: | :---: |
| <5 | 优秀 |
| 5~25 | 一般 |
| >25 | 较差 |
| >100 | 异常 |

溶解性总固体 (TDS)

| TDS (mg/L) | 评价 |
| :---: | :---: |
| <300 | 优秀 |
| 300~500 | 良好 |
| 500~1000 | 一般 |
| >1000 | 较差 |
